# FlyRank Backend AI Engineering API

A minimal, beginner-friendly **FastAPI** backend built for the **FlyRank Backend AI Engineering** assignment.

The project intentionally stays as small as possible while still following professional
backend engineering conventions: a clean project layout, type hints, docstrings/comments,
a `requirements.txt`, a `.gitignore`, an MIT license, and automatic Swagger documentation.

---

## 📋 Project Description

This is a tiny web server exposing exactly two JSON endpoints:

| Method | Endpoint    | Description                                   |
|--------|-------------|------------------------------------------------|
| GET    | `/`         | Returns a simple greeting message.             |
| GET    | `/profile`  | Returns basic information about the developer. |

It is built with **FastAPI**, a modern, high-performance Python web framework, and served
with **Uvicorn**, an ASGI server. FastAPI also gives us **automatic interactive API docs**
for free at the `/docs` route.

## ✅ Prerequisites

Before you start, make sure you have:

- **Python 3.9+** installed. Check with:
  ```bash
  python3 --version
  ```
- **pip** (Python's package manager), which usually comes bundled with Python.
- (Optional but recommended) **git**, if you plan to clone/push this repository.

## 🐍 Virtual Environment Setup

It's best practice to isolate project dependencies inside a virtual environment
instead of installing them globally.

### Create the virtual environment
```bash
python3 -m venv venv
```

### Activate it

**On macOS / Linux:**
```bash
source venv/bin/activate
```

**On Windows (PowerShell):**
```powershell
venv\Scripts\Activate.ps1
```

**On Windows (Command Prompt):**
```cmd
venv\Scripts\activate.bat
```

Once activated, you should see `(venv)` at the beginning of your terminal prompt.

## 📦 Installation Instructions

With your virtual environment activated, install the project dependencies:

```bash
pip install -r requirements.txt
```

This installs:
- `fastapi` — the web framework used to build the API.
- `uvicorn[standard]` — the ASGI server used to run the API.

---

## 🚀 How to Run the Server

From the **project root** (the folder containing the `app/` directory), run:

```bash
uvicorn app.main:app --reload
```

Explanation of this command:
- `app.main` → refers to the file `app/main.py`.
- `app.main:app` → tells Uvicorn to look for the variable named `app` (our FastAPI instance) inside that file.
- `--reload` → automatically restarts the server whenever you change the code (great for development).

By default, the server will start at:
```
http://127.0.0.1:8000
```

---

## 🌐 Browser URLs

Once the server is running, open any of these URLs in your browser:

- Root endpoint:
  http://127.0.0.1:8000/

- Profile endpoint:
  http://127.0.0.1:8000/profile

- Interactive Swagger documentation (auto-generated by FastAPI):
  http://127.0.0.1:8000/docs

- Alternative ReDoc documentation:
  http://127.0.0.1:8000/redoc
